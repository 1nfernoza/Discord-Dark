# Discord-Dark

	A Github Dark Theme for Discord
	
	Maintained by: SyndiShanX
	Inspired by: Github Dark
	This Theme is meant to be used with Discord Dark Mode
	
	Installation:
	Copy discorddark.theme.css into your Better Discord Themes Folder
